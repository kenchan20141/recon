隱私權與資安聲明

1. Edge Computing Technology (邊緣運算技術)

This system uses local browser-based processing. No video stream is ever sent to a server.

本系統使用瀏覽器端運算技術。影像畫面絕不會上傳至任何伺服器。

2. Data Collection (資料收集)

Only timestamps of potential violations are logged. No images are stored.

僅記錄違規事件的時間點，不儲存影像。

3. Security Architecture (資安防護架構)

  • Input Sanitization: All user inputs are sanitized.
  • CSP: Strict browser policies block unauthorized scripts.
